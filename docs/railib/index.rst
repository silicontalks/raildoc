====================
RAIL Library
====================

The release of RAIL libary consists of analog and digital library. 
The analog library is virtuoso based, including the RAIL cells' schematics, symbols, layouts and testbenches.
The digital library is milkyway based, including a db file, and CEL/FRAM library compatible for backend place and routing tools.
Note that gds/CEL files are only available for those used already signed NDA, please contract rail4open@gmail.com

All available library are accessible via the link: https://github.com/rail-posh .
